Project Overview:
  - Unity Simulation
    Developed a 3D robotic arm environment with inverse kinematics for smooth, target-driven motion.
  - ROS 2 Integration
    Implemented a modular communication layer for handling robot state, control signals, and simulation data flow.
  - Gazebo & RViz
    Simulated the robotic arm in a physics-based environment and visualized joint states, motion, and sensor feedback.
  - TCP/IP Communication
    Built a bidirectional interface between Unity and Python to stream real-time positional data and control signals.
  - ECG Sensor Interface (AD8232)
    Processed biosignals into discrete control commands (-1, 0, 1) to influence robotic arm movement.
